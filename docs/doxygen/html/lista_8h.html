Busca un valor y copia posiciones de coincidencia.

Busca un valor y copia posiciones de coincidencia.

Recorre toda la lista comparando cada nodo con valor. Cada vez que encuentra una coincidencia, si destino no es NULL y queda capacidad, almacena la posición del nodo. El conteo siempre se realiza aunque destino sea NULL o se haya llenado el arreglo.

Parámetros
[in]listaPuntero constante a la Lista.
[in]valorValor entero a buscar.
[out]destinoArreglo donde se almacenan las posiciones encontradas. Puede ser NULL si solo se desea contar.
[in]capacidadTamaño máximo del arreglo destino.
Devuelve
Número total de coincidencias encontradas en la lista. Retorna 0 si lista es NULL.
Nota
Complejidad temporal: O(n).

Copia valores de la lista en un arreglo destino.

Copia valores de la lista en un arreglo destino.

Recorre la lista desde cabeza copiando el valor de cada nodo en destino. El índice 0 corresponde al primer nodo (cabeza). La copia se detiene cuando se agota la lista o se alcanza capacidad. La lista no se modifica.

Parámetros
[in]listaPuntero constante a la Lista de origen.
[out]destinoArreglo donde se almacenarán los valores copiados.
[in]capacidadNúmero máximo de elementos que caben en destino.
Devuelve
Número de elementos efectivamente copiados. Retorna 0 si lista, destino son NULL o capacidad <= 0.
Nota
Complejidad temporal: O(min(n, capacidad)).

Elimina la primera ocurrencia de un valor.

Elimina la primera ocurrencia de un valor.

Recorre la lista con un puntero p al nodo actual y ant al anterior. Cuando encuentra un nodo con nro == valor, actualiza el enlace de ant (o de cabeza si es el primero) y libera el nodo. Se detiene tras la primera coincidencia.

Parámetros
[in,out]listaPuntero a la Lista.
[in]valorValor entero a eliminar.
Devuelve
true si se encontró y eliminó el nodo.
false si lista es NULL o el valor no existe.
Postcondición
El tamaño de la lista disminuye en 1 si retorna true.
Nota
Complejidad temporal: O(n).

void lista_formatear ( const Lista * lista,
char * destino,
size_t capacidad )

Genera una representacion textual de la lista.

Genera una representacion textual de la lista.

Escribe en destino una cadena donde cada elemento aparece como "[pos]=valor" separados por " -> ". Ejemplo:

[1]=10 -> [2]=20 -> [3]=30

Si la lista está vacía escribe "Lista vacia". La cadena siempre queda terminada en '\0' si capacidad >= 1.

Parámetros
[in]listaPuntero constante a la Lista a representar.
[out]destinoBuffer de caracteres donde se escribirá el resultado.
[in]capacidadTamaño en bytes del buffer destino.
Precondición
destino debe apuntar a un buffer de al menos capacidad bytes.
Postcondición
destino contiene una cadena terminada en '\0'.
Nota
Si el contenido es más largo que capacidad, se trunca de forma segura (usa snprintf internamente).
Si destino es NULL o capacidad es 0, la función no hace nada.
Complejidad temporal: O(n).

void lista_invertir ( Lista * lista)

Invierte el orden de los nodos.

Invierte el orden de los nodos.

Aplica el algoritmo clásico de tres punteros (prev, curr, next) para invertir los enlaces in-place sin reservar memoria adicional. Al terminar, la antigua cabeza pasa a ser el último nodo y el antiguo último nodo pasa a ser la nueva cabeza.

Parámetros
[in,out]listaPuntero a la Lista a invertir. Si es NULL la función no hace nada.
Nota
Complejidad temporal: O(n).
Complejidad espacial: O(1) (no reserva memoria).

bool lista_orden_asc ( const Lista * lista)

Verifica si la lista esta en orden ascendente.

Verifica si la lista esta en orden ascendente.

Recorre la lista comparando cada nodo con su sucesor. Si en algún punto el valor de un nodo es mayor que el del siguiente, retorna false inmediatamente. Una lista vacía o de un solo elemento se considera ordenada (retorna true).

Parámetros
[in]listaPuntero constante a la Lista.
Devuelve
true si todos los elementos están en orden no decreciente, o si la lista tiene 0 ó 1 elementos.
false si hay al menos un par fuera de orden o lista es NULL.
Nota
Complejidad temporal: O(n).
