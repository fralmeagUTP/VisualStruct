Descripción detallada

Implementación del TAD Grafo con algoritmos clásicos.

Implementa grafos dirigidos y no dirigidos usando lista de adyacencia. Los vértices se almacenan en una lista enlazada, y cada vértice tiene una lista enlazada de aristas salientes.
