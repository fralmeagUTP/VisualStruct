Descripción detallada

Algoritmo de layout circular para posicionamiento de vértices.

Autor
VisualStruct v2

Calcula las posiciones de vértices en disposición circular o fuerza dirigida. Independiente de Raylib (solo usa matemática).

bool grafo_layout_calcular_circular ( GrafoState * estado,
const GrafoLayoutConfiguracion * config )

Calcula posiciones circulares de vértices.

Distribuye vértices uniformemente en un círculo dentro del panel. Radio del círculo se adapta al tamaño del panel y cantidad de vértices.

Parámetros
estadoPuntero al estado visual (se modifican x, y)
configConfiguración de layout
Devuelve
true si se calculó exitosamente
