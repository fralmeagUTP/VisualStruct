Descripción detallada

Estado visual del grafo para animaciones y visualización.

Autor
VisualStruct v2

Maneja el estado visual de vértices y aristas durante la ejecución de algoritmos. Limita a 64 vértices y 256 aristas para optimizar renderizado en tiempo real. No depende de Raylib (independencia de capas de presentación).
